The following file contains the work done for the final assignment of the course Fundamentals of Data Science. The assignment aim is to present an analysis of crime incidents during the COVID-19 pandamic in the Netherlands, UK, and certain cities accross the US. Various data collection method has been used during the execution of this project.
The final outcome is a descriptive analysis presented in an interactive dashboard using Plotly.
For the theoratical framework, and further details of the project please download document "Extended Abstract.docx".
